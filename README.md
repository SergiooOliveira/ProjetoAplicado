 # Redes

 O jogo oferece a possibilidade de jogar modo cooperativo, o que torna a experiência mais interessante e divertida. O modo cooperativo promove a interação entre os jogadores, aumentando a dinâmica e o desafio do jogo.
	A arquitetura de rede adotada será do tipo Host-Server. Neste modelo, um dos jogadores será o host da partida, enquanto os outros jogadores poderão entrar na sessão do host, participando na mesma partida. Desta forma não será necessário um servidor dedicado para hospedar as partidas.

  # Serviços Multiplayer

  A escolha final foi o Photon. Após muitas alteções de qual serviço utilizar, entre Fishnet, PurrNet e Photon.
  O Photon oferece a possibilidade de Host, oferecendo servidores para desenvolvimento de projeto gratuitamente, se passar para produção tem custo adicionais.

   # Dificuldades
